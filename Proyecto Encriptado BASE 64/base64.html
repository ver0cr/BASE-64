¿Cómo se usa?


El método de codificación Base64 requiere que cada tres bytes de 8 bits se conviertan en cuatro bytes de 6 bits. Entre estos cuatro bytes después de la conversión, cada seis bits válidos son datos válidos, y los dos bits restantes usan 0


Origen


La intención original de Base64 era cumplir con el requisito de que los caracteres no ASCII no se pueden usar directamente en los correos electrónicos porque algunos de los primeros servidores de correo solo admiten mensajes de texto, no mensajes y archivos binarios. El escenario de aplicación más utilizado es transmitir mensajes de texto largos a través del protocolo http. Empaqueta los datos binarios ingresados ​​por el usuario en un formato seguro y los envía como el valor del campo de encabezado http sin preocuparse de que contenga dos puntos, saltos de línea o valores binarios que destruirán el programa de análisis HTTP.
